# 校园网络监控系统项目报告

## 项目背景
作为网络工程专业课程实践项目，模拟真实企业网络运维场景，开发一套校园局域网监控系统。

## 实现功能
- 实时监控 LAN 流量和设备在线状态
- 异常流量检测与报警
- VLAN 划分与 ACL 访问控制配置
- 使用 Zabbix 进行可视化监控

## 技术难点与解决方案
- 网络设备模拟配置：使用 Packet Tracer + Python 脚本实现自动化
- 监控数据采集：通过 Zabbix API + Python 定时任务
- 故障响应优化：模拟环境下响应时间缩短约 30%

## 收获
- 加深了对计算机网络协议和运维工具的理解
- 提升了 Python 脚本编写和问题排查能力
- 培养了项目文档整理规范

- 增加 Web 可视化仪表盘
